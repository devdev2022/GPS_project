# GPS_project

## 기능
1. 운전자와 유저의 회원가입과 로그인
2. 유저의 예약조회(JWT를 이용한 인증방식)
3. 유저의 예약요청(JWT를 이용한 인증방식) 
4. 운전자의 예약수락(JWT를 이용한 인증방식) 
5. 운전자의 예약검색(JWT를 이용한 인증방식)
